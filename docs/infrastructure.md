# Infrastructure

## Backend
- Language: Python
- Web Framework: FastAPI
- Database: PostgreSQL
- ORM: SQLAlchemy
- Real Time Data: WebSockets via FastAPI
- Authentication: JWT
- Testing: pytest
- Caching: Redis
- Background Tasks: Celery

## Frontend
- Language: TypeScript
- Web Framework: React + React Query
- UX: TailwindCSS / MaterialUI
- Real Time Data: WebSockets via FastAPI
- Visualisation: Chart.js
- Routing: React Router
